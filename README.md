# MyPlan - 智能待办事项管理应用

一个简洁、优雅、美观的纯前端待办事项Web应用程序，采用现代化设计和高级配色方案。

## 🌟 功能特性

### 核心功能
- **四级优先级分类**：重要、每日、次要、一般
- **本地数据持久化**：自动保存，刷新不丢失
- **任务管理**：添加、编辑、删除、完成标记
- **智能过滤**：按优先级筛选任务
- **实时统计**：总任务数和完成数统计

### 用户体验
- **响应式设计**：完美适配桌面和移动设备
- **优雅动画**：流畅的交互动效
- **快捷键支持**：提升操作效率
- **通知系统**：操作反馈提示
- **数据导入导出**：备份和恢复功能

## 🎨 设计特色

### 视觉设计
- **现代化界面**：采用卡片式布局和圆角设计
- **高级配色**：深蓝紫渐变主题色调
- **优先级色彩**：

### 交互体验
- **悬停效果**：鼠标悬停时的视觉反馈
- **加载动画**：平滑的页面过渡
- **模态框编辑**：优雅的任务编辑体验
- **拖拽友好**：为未来功能预留接口

## 🚀 快速开始

### 方法一：直接打开
1. 下载所有文件到本地目录
2. 双击 `index.html` 文件
3. 在浏览器中打开应用

### 方法二：本地服务器（推荐）
```bash
# 进入项目目录
cd MyPlan

# 启动Python服务器
python3 -m http.server 8000

# 或使用Node.js服务器
npx serve .

# 在浏览器中访问
http://localhost:8000
```

## 📖 使用指南

### 基本操作

#### 添加任务
1. 在输入框中输入任务内容
2. 选择优先级（重要/每日/次要/一般）
3. 点击 ➕ 按钮或按 Enter 键

#### 管理任务
- **完成任务**：点击任务前的圆圈复选框
- **编辑任务**：点击任务右侧的编辑按钮 ✏️
- **删除任务**：点击任务右侧的删除按钮 🗑️

#### 过滤查看
- 点击顶部的过滤标签：全部、重要、每日、次要、一般
- 快速查看特定优先级的任务

### 快捷键
- `Ctrl/Cmd + N`：聚焦到任务输入框
- `Ctrl/Cmd + E`：导出任务数据
- `Enter`：在输入框中添加任务
- `Esc`：关闭编辑模态框

### 数据管理

#### 自动保存
- 所有操作自动保存到浏览器本地存储
- 页面刷新后数据不会丢失
- 支持离线使用

#### 备份与恢复
- **导出**：使用 `Ctrl/Cmd + E` 导出JSON格式备份文件
- **导入**：通过文件选择器导入备份数据
- **清空**：可一键清空所有任务数据

## 🎯 优先级系统

### 分类说明
- **重要**：紧急且重要的任务，需要优先处理
- **每日**：日常例行任务，需要定期完成
- **次要**：重要但不紧急的任务，可以安排时间处理
- **一般**：普通任务，时间充裕时处理

### 使用建议
1. **重要任务**：工作截止日期、重要会议等
2. **每日任务**：锻炼、学习、日常工作等
3. **次要任务**：技能提升、长期项目等
4. **一般任务**：整理、购物、娱乐等

## 🐛 故障排除

### 常见问题

#### 数据丢失
- 检查浏览器是否启用了LocalStorage
- 确认没有使用隐私/无痕模式
- 尝试导入之前的备份文件

#### 样式异常
- 确保所有CSS文件正确加载
- 检查浏览器兼容性
- 清除浏览器缓存

#### 功能异常
- 打开浏览器开发者工具查看错误信息
- 确保JavaScript已启用
- 检查文件路径是否正确

- ✅ Firefox 55+
- ✅ Safari 12+
- ✅ Edge 79+

## 📈 性能优化

### 已实现的优化
